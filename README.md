# Midterm Project - Task Management Application
ITC 4214 - Internet Programming A1 | Spring 2026 | Maria Georgia Krousi

## About the Project:
Mock Task Management Application for an indie game development company, allowing users to organize and view tasks while also providing basic information about the company.<br><br>
Goal of this project is to showcase skill in client-side and server side languages for the web & effective use of libraries and frameworks.
